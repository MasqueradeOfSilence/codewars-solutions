<h2><a href=https://www.codewars.com/kata/56d5166ec87df55dbe000063/train/python target="_blank">Sum of Array Averages</a></h2><h3>7 kyu</h3><p>Program a function <code>sumAverage(arr)</code> where <code>arr</code> is an array containing arrays full of numbers.</p><p>First, determine the average of each array. Then, return the sum of all the averages.</p><ul><li>All numbers will be less than 100 and greater than -100.</li><li><code>arr</code> will contain a maximum of 50 arrays.</li><li>After calculating all the averages, add them <strong>all</strong> together, <strong>then</strong> round down, as shown in the example below:</li></ul><p>The example given: <code>sumAverage([[3, 4, 1, 3, 5, 1, 4], [21, 54, 33, 21, 77]])</code>, the answer being 44.</p><ol><li>Calculate the average of each individual array:<pre><code>[3, 4, 1, 3, 5, 1, 4] = (3 + 4 + 1 + 3 + 5 + 1 + 4) / 7 = 3[21, 54, 33, 21, 77] = (21 + 54 + 33 + 21 + 77) / 5 = 41.2</code></pre></li><li>Add the average of each array together:<pre><code>3 + 41.2 = 44.2</code></pre></li><li>Round the final average down:<pre><code>floor(44.2) = 44</code></pre></li></ol>
